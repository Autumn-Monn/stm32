# STM32F103C8T6引脚分配

    下面这版分配比较均衡，也适合 CubeMX 配置：

* PA0：**ADC1_IN0**，接土壤湿度模块 FC-28 AO
* PA1：GPIO 输出，继电器 IN1，控制水泵
* PA2：GPIO 输出，继电器 IN2，控制风扇
* PA3：GPIO 输出，蜂鸣器模块
* PA4：GPIO，DS18B20 Data
* PA8：GPIO 输出，绿灯
* PA9：**USART1_TX**，接 ESP8266 RX
* PA10：**USART1_RX**，接 ESP8266 TX
* PB0：GPIO 输出，红灯
* PB1：GPIO 输出，蓝灯
* PB6：**I2C1_SCL**
* PB7：**I2C1_SDA**
* PB12：按键 1，GPIO 输入上拉
* PB13：按键 2，GPIO 输入上拉
* PB14：按键 3，GPIO 输入上拉
* PB15：按键 4，GPIO 输入上拉
