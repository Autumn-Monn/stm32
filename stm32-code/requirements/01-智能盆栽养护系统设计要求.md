# **题目二：智能盆栽养护系统设计**

## **一、设计要求**

### **1、功能要求**

#### （1）环境监测

1. 实时采集土壤湿度、土壤温度。
2. 数据精度：温度±0.5℃，土壤湿度±3%。

#### （2）自动化控制

1. 自动灌溉：土壤湿度低于下限阈值启动水泵浇水，达标停止。
2. 环境调节：土壤温度超上限启动风扇或遮阳帘。
3. 手动模式：按键强制启停水泵、风扇。

#### （3）用户交互

1. OLED显示土壤温湿度、系统状态。
2. 三色LED指示：红色缺水、蓝色低温、绿色正常。
3. 土壤极端干燥或超温时蜂鸣器报警。
4. 支持按植物类型自适应调整阈值。

#### （4）远程监控（创新拓展）

    蓝牙/Wi‑Fi+MQTT上传数据至手机APP。

#### （5）节能与数据管理（创新拓展）

1. 低功耗：休眠+定时唤醒。
2. 历史数据存储：温湿度、灌溉记录。

### **2、性能指标**

1. 传感器采集精度达标。
2. 控制响应延迟＜1秒。
3. 系统稳定无漏控、误触发。

## **二、设计方案**

### **1、系统架构**

**采用物联网分层架构**

1. 感知层：土壤温湿度传感器采集数据。
2. 网络层：Wi‑Fi/蓝牙无线传输。
3. 控制层：主控逻辑处理与执行器驱动。
4. 应用层：本地显示、远程监控、数据存储。

### **2、方案选择**

（1）主控芯片：推荐ESP32；备选STM32、Arduino。

（2）传感器：土壤湿度传感器、DS18B20。

（3）执行器件：微型水泵、风扇。

（4）通信：Wi‑Fi/蓝牙+MQTT。

（5）供电：USB或锂电池。

## **三、硬件模块**

（1）主控芯片：STM32103C8T6

（2）土壤湿度传感器：FC‑28 + LM393模块

（3）温度传感器：DS18B20（带不锈钢探针）

（4）执行模块：水泵、风扇（由双路5V继电器模块控制）

（5）显示模块：0.96寸OLED屏幕

（6）指示模块：三色LED(0805贴片)

（7）报警模块：有源蜂鸣器(直插，高电平有效)

（8）交互模块：设置按键、模式按键

（9）电源模块：5V锂电池

（10）WIFI模块：ESP8266（直插）

（10）其他元件：

* 3路继电器模块
* 独立贴片按键×4
* 5V电源接口
